Características del servidor

1. Definición:

  • Active Directory
  • Servidor DNS
  • Windows Defender
  • Servicio de Escritorio Remoto
  • Web Application Proxy
  • Failover Clustering
  • Reinicio de software

2. Justificación:

Hemos decidido escoger Windows Server 2016 como sistema operativo para el servidor, ya que se nos hace mucho más sencillo trabajar con Windows Server 2016 porque trabajar gráficamente nos ayuda a aclararnos a la hora de tener que trabajar. Pese a estar desactualizado, contiene una lista de errores limitados y la solución de estos. Además, es la versión que utilizamos en M04 (Sistemes operatius en xarxa).

 

Características del cliente

1. Definición:

  • Windows 10 Profesional
  • Interfaz de usuario
  • Menú de inicio
  • Asistente virtual Cortana
  • Escritorio remoto
  • Creación y unirse a dominio
  • Configuración compartida
  • Windows Defender

2. Justificación:

Hemos elegido Windows 10 Pro como sistema operativo principal para nuestro cliente. Es un sistema operativo que hemos utilizado mucho para nuestro uso personal y nos sentimos cómodos utilizándolo.

Su interfaz nos facilita la navegación por el SO porque es muy intuitiva. El menú de inicio, nos facilita muchas opciones como un índice de aplicaciones, opciones de apagado, recomendaciones de Microsoft, etc…
